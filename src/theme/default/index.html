虚拟网络概念 方便团队管理和销售

LightNet VPN 抽象出一层网络的概念, 网络有网络号和序列号组成, 一个网络下辖多个序列号, 由1个网络号+1个序列号组成一个唯一联网的凭证, 网络的所有者可以对下面的序列号进行管理和销售.
